# FinAdvisor

## Problem Statement
Many citizens, especially rural users and elderly people, lack digital financial literacy.

## Solution
FinAdvisor is an AI-powered financial assistant built using IBM watsonx.ai that provides simple guidance on:
- UPI
- Net Banking
- Loans
- Budgeting
- Government Schemes
- Investments
- Fraud Prevention

## Technologies Used
- IBM watsonx.ai
- Granite Foundation Model
- Prompt Engineering
- IBM Cloud

## Features
- Beginner-friendly explanations
- Multilingual support
- Financial safety tips
- Government scheme information

## Future Scope
- Voice assistant support
- Regional language support
- Personalized budgeting tools
